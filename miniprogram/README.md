# 微信小程序项目

这是微信小程序的根目录，请用微信开发者工具打开此目录。

## 📁 目录结构

```
miniprogram/                  # 小程序根目录
├── app.js                   # 应用主入口
├── app.json                 # 全局配置
├── app.wxss                 # 全局样式
├── project.config.json      # 项目配置
├── sitemap.json            # 搜索配置
├── pages/                  # 页面目录
│   └── index/              # 首页
├── components/             # 自定义组件
├── utils/                  # 工具函数
├── config/                 # 配置文件
├── assets/                 # 静态资源
│   ├── images/            # 图片资源
│   └── icons/             # 图标资源
└── cloud/                  # 云开发
    └── functions/          # 云函数
```

## 🚀 快速开始

1. 用微信开发者工具打开此目录（miniprogram）
2. 配置AppID
3. 开始开发

## 🔧 技术栈

- 微信小程序原生框架
- 微信云开发
- TypeScript支持
- Less样式预处理
